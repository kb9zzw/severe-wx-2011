2011 Tornadoes

Overview

This interactive map shows 2011 tornado track information, color coded by Fujita scale. Mouse over a tornado track to show summary information. Tornadoes with fatalities are indicated with a red dot.

About this project

This website was created by Jon Burroughs as an assignment for Computer Cartography (GIS 515) at North Carolina State University.

Sources for data, photographs were provided by NOAA, specifically the following sites:

  • Storm Prediction Center - GIS Data
  • National Hurricane Center - GIS Data
  • NOAA Extreme Weather 2011
  • National Weather Service

 

Technologies used in the creation of this project, include:

  • MapBox - used for interactive mapping
  • Google Drive - used for charting
  • ESRI ArcGIS - used for initial data processing
  • Slidorion - used for image slider
  • FreeCSSTemplates - used for page layout and design
